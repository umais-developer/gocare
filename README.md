# GoCare - Coming Soon Landing Page

A React + Vite static SPA using the Stylish Portfolio Bootstrap theme. A Coming Soon landing page for GoCare healthcare services.

## Features

- ✨ Live countdown timer to launch date (September 1, 2026)
- 📧 Email notification signup form
- 📱 Fully responsive design
- 🎨 Stylish Portfolio Bootstrap theme
- ⚡ Built with Vite for fast development and optimized builds
- 🚀 GitHub Pages deployment via GitHub Actions

## Tech Stack

- React 18
- Vite 6
- Bootstrap 5
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173/gocare/` in your browser.

### Build

```bash
npm run build
```

The build output will be in the `dist` directory.

### Preview

```bash
npm run preview
```

## Deployment

This project is deployed to GitHub Pages via GitHub Actions. Push to the `main` branch to trigger automatic deployment.

**GitHub Pages URL:** https://umais-developer.github.io/gocare/
